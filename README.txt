** Como utilizar
Para executar o programa, basta executar o arquivo "8-puzzle.exe" presente na raiz do projeto.
Se preferir, poderá executá-lo pelo cli da linguagem Go com o comando `go run main.go`.
Também é possível compilar o código pelo comando `go build` e executar o binário `8-puzzle` ou `8-puzzle.exe` que será gerado na raiz do projeto.

Ao iniciar o programa, serão solicitadas as linhas que compõe o tabuleiro inicial.
Elas devem ser inseridas separadamente, uma após a outra, com um espaço entre cada número. Por exemplo: `7 8 0`
OBS: o número 0 é utilizado para representar o espaço vazio no tabuleiro

** Sobre o programa desenvolvido
Desenvolvido utilizando a linguagem Go.
O código-fonte pode ser consultado no arquivo "main.go" ou através do repositório https://github.com/mpedroni/8-puzzle
